# claude-profile documentation

`claude-profile` launches `claude` with only the plugins, skills, marketplaces, and MCP servers
a profile defines — vendored into that profile's own directory and never installed into your
real `~/.claude`. These pages are the full reference for authoring, running, and understanding
profiles.

- [Authoring profiles](profiles.md) — the profile JSON format, marketplaces, pinning, `extends`.
- [Command reference](commands.md) — every command, its flags, and behavior.
- [How it works](how-it-works.md) — the isolation model, provisioning, pinning, and known limitations.
- [Statusline snippet](statusline.md) — show the active profile in your Claude Code statusline.

For a high-level overview, installation, and uninstallation, see the [top-level README](../README.md).
